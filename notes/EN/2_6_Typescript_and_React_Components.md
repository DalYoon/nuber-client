### What is 'React.SFC'

- `SFC` stands for Stateless Functional Component

```js
import React from "react";

const ThisComponent = props => {
  return <div>This is SFC</div>;
};
```

or

```js
import React from "react";

function ThisComponent(props) {
  return <div>This is SFC</div>;
}
```
